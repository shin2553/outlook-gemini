# VBA 리본 버튼 설치 방법

## 1. VBA 코드 등록

1. Outlook 열기
2. `Alt + F11` → VBA 편집기 실행
3. 왼쪽 트리에서 `Project1 > Microsoft Outlook 개체 > ThisOutlookSession` 더블클릭
4. `GeminiReply.bas` 파일의 내용을 붙여넣기
5. `Ctrl + S` 저장

## 2. 리본 버튼 추가 (빠른 실행 도구 모음)

가장 간단한 방법:

1. Outlook 상단 빠른 실행 도구 모음 우클릭 → **빠른 실행 도구 모음 사용자 지정**
2. 명령 선택: **매크로**
3. `GeminiAutoReply` 선택 → **추가**
4. 확인

## 3. 사용 방법

1. Outlook에서 답장할 메일을 선택하거나 열기
2. 빠른 실행 도구 모음의 **GeminiAutoReply** 버튼 클릭
3. "생성 중" 팝업 확인
4. 약 5~15초 후 답장 창이 자동으로 열림
5. 초안 검토 후 발송

## 주의사항

- 메일이 선택되지 않은 상태에서 실행하면 오류 메시지 표시
- 오류 발생 시 `C:\Users\82108\Downloads\outlook_gemini\outlook_gemini.log` 확인
