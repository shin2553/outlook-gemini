# ReadMe_SleepMode

[p.1]
SCANLAB GmbH, 2018-03-20 
 
Script SleepMode.cmd 
 
Description 
The SleepMode.cmd script deactivates the standby and hibernate mode by changing 
the system registry’s corresponding power settings, if necessary. Notice that this kind 
of deactivation also disables the start menu’s options to manually switch to any 
power saving mode. 
 
Usage 
Right-click the SleepMode.cmd script to run it with administrative privileges. After-
wards restart Windows. 
 
 
 
Note 
 
The changed power setting takes effect after a restart.
