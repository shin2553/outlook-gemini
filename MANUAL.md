## 1. 시스템 요구사항

| 항목 | 요구사항 |
|---|---|
| 운영체제 | Windows 10 / 11 (64bit) |
| Outlook | Microsoft Office 2016 이상 (독립 설치형 또는 Microsoft 365) |
| 인터넷 | Gemini API 호출을 위한 인터넷 연결 필수 |
| Gemini API 키 | Google AI Studio에서 발급 필요 (무료) |

> Python 설치 불필요. Outlook과 인터넷 연결만 있으면 됩니다.

## 2. 프로그램 설치

1. 배포받은 `OutlookGemini` 폴더를 원하는 위치에 복사합니다
   (예: `C:\Program Files\OutlookGemini\` 또는 바탕화면)

2. 폴더 구성 확인:

```
OutlookGemini\
├── OutlookGemini.exe   ← 실행 파일
├── config.ini          ← 설정 파일 (API 키, 경로 등)
├── profiles.json       ← 담당자 프로필
└── MANUAL.md           ← 이 매뉴얼
```

3. `OutlookGemini.exe`를 더블클릭하여 실행
4. 첫 실행 시 [최초 설정](#3-최초-설정) 진행

> **바로가기 생성:** `OutlookGemini.exe`에서 우클릭 → "바로 가기 만들기" → 바탕화면으로 이동

## 3. 최초 설정

프로그램 첫 실행 후 우측 상단 **[설정]** 버튼을 클릭합니다.

### 3-1. Gemini API 키 설정

API 키는 Gemini AI를 사용하기 위해 반드시 필요합니다.

1. 설정 창 **[Gemini API 키]** 탭 선택
2. API 키가 없는 경우 탭 안의 **"Google AI Studio에서 발급하기 →"** 링크 클릭
   → 브라우저에서 발급 페이지로 바로 이동합니다
3. Google 계정으로 로그인 후 **[Create API key]** 클릭하여 키 발급
4. 발급된 키를 복사하여 **API 키** 입력란에 붙여넣기
5. **[키 표시]** 체크박스로 입력 내용 확인 가능
6. **[연결 테스트]** 버튼으로 정상 작동 확인
7. **[저장]** 클릭

### 3-2. 경로 설정

프로그램이 매뉴얼 파일을 찾을 수 있도록 경로를 지정합니다.

1. 설정 창 **[매뉴얼 관리]** 탭 선택
2. 아래 항목들의 경로를 설정합니다:

| 항목 | 설명 | 예시 |
|---|---|---|
| **txt 매뉴얼 폴더** | 텍스트로 변환된 매뉴얼이 있는 폴더 | `C:\scanlab_support\NotebookLM_exports` |
| **PDF 원본 폴더** | PDF 매뉴얼 원본이 있는 폴더 (선택사항) | `C:\scanlab_support\Manuals` |

3. 각 항목 옆 **[찾아보기]** 버튼으로 폴더 선택
4. **[저장]** 클릭

### 3-3. 회사 정보 설정

Gemini가 답변을 작성할 때 사용하는 회사 소개입니다.

1. 설정 창 **[회사 정보]** 탭 선택
2. **관계 설명** 란에 회사 역할 입력
   (예: `여러 제조사 제품의 국내 공식 대리점`)
3. **[저장]** 클릭

## 4. 답변 주체 프로필 관리

이메일 초안에 서명처럼 들어갈 담당자 정보를 관리합니다.

### 프로필 추가

1. 메인 화면 상단 **[프로필 관리]** 버튼 클릭
2. 팝업 창에서 **[추가]** 버튼 클릭
3. 이름, 직책, 회사, 부서 입력
4. **[저장]** 클릭

### 프로필 선택

- 메인 화면 상단 드롭다운에서 원하는 프로필 선택
- 선택 즉시 이름 / 직책 / 회사 / 부서 필드가 자동으로 채워집니다

### 기본 프로필 지정

프로그램 시작 시 자동으로 선택되는 프로필을 지정하려면:

1. **[프로필 관리]** 버튼 클릭
2. 원하는 프로필 선택 후 **[기본값으로 설정]** 클릭 (또는 편집 창에서 설정)

## 5. 매뉴얼 시스템 구성

이 프로그램은 제품 매뉴얼 파일을 기반으로 답변을 생성합니다.
매뉴얼은 **텍스트(txt)** 와 **PDF 원본** 두 가지 형태로 관리됩니다.

### 5-1. 텍스트 매뉴얼(txt)

- PDF를 텍스트로 변환한 파일들이 저장되는 폴더
- 파일명 예: `RTC5_manual.txt`, `SCANmotionTCP_guide.txt`
- **매뉴얼 검색의 기본 단위** — 반드시 준비되어 있어야 합니다

### 5-2. PDF 원본 매뉴얼

- PDF 파일이 저장되는 폴더
- 텍스트 매뉴얼과 **동일한 파일명(확장자만 다름)** 으로 저장 권장
  - 예: `RTC5_manual.pdf` ↔ `RTC5_manual.txt`
- PDF 폴더 없이도 텍스트 전용 모드로 동작 가능
- PDF가 있으면 도면, 표, 그래프 등 시각 정보 분석이 가능합니다

### 5-3. PDF → txt 일괄 변환

새 PDF 매뉴얼을 추가했을 때 텍스트 변환이 필요합니다.

1. 설정 창 **[매뉴얼 관리]** 탭 클릭
2. **"미변환 PDF 파일"** 목록 확인 (txt 파일이 없는 PDF가 자동으로 표시됨)
3. 변환할 파일을 체크 후 **[선택 변환]** 또는 **[전체 자동 변환]** 클릭
4. 변환 완료 후 txt 파일이 텍스트 매뉴얼 폴더에 자동 저장됨

> **주의:** 스캔 이미지 기반 PDF는 텍스트 추출이 불가능합니다.
> 이 경우 Adobe Acrobat 등 외부 OCR 도구로 먼저 변환 후 사용하거나,
> PDF 직접 분석 모드를 사용하세요 (Gemini가 이미지로 직접 읽음).

## 6. 기본 사용법 — 단계별 워크플로우

### Step 1 — Outlook에서 답변할 메일 선택

- Outlook에서 답변할 고객 문의 메일을 **열거나 선택**합니다
- 메일을 열어 두거나 받은편지함에서 클릭하여 선택한 상태로 유지

### Step 2 — 프로필 선택

- 메인 화면 상단 드롭다운에서 **답변 작성자 프로필** 선택

### Step 3 — 메일 불러오기

- **[메일 불러오기]** 버튼 클릭
- Outlook에서 현재 선택/열린 메일의 내용이 자동으로 추출됩니다
- 화면에 표시되는 정보:
  - 발신자 / 제목 / 날짜
  - 본문 미리보기
  - 첨부파일 목록 (이미지 / PDF)
  - 매칭된 매뉴얼 파일명 목록

### Step 4 — 분석 모드 선택 (선택사항)

- **자동 모드** (기본값 권장): 텍스트 분석 후 시각 정보가 필요하면 PDF로 자동 전환
- **텍스트 전용**: 빠른 분석, 텍스트 매뉴얼만 사용
- **PDF 직접 분석**: 정확한 분석, 처음부터 PDF 원본 사용 (느림)

### Step 5 — 추가 지시사항 입력 (선택사항)

- 자유 텍스트로 특별 요청사항 입력
- 예: `답변에 구체적인 설정값 예시를 포함해주세요`

### Step 6 — 초안 생성

- **[초안 생성]** 버튼 클릭
- 진행 상태 바와 상태 메시지가 표시됩니다 (수초~수십초 소요)
- 생성 완료 시 두 영역에 결과가 표시됩니다:
  - **좌측 — 문의 분석**: 문의 요약, 관련 매뉴얼 근거, 답변 방향, 주의 사항
  - **우측 — 이메일 초안**: 바로 발송 가능한 형태의 이메일 (직접 편집 가능)

### Step 7 — 검토 및 편집

- 우측 이메일 초안 텍스트는 **직접 수정 가능**
- `[추정]` 태그: 일반 기술 지식으로 보충한 내용 — 검토 후 확인
- `[확인 필요]` 태그: 불확실한 내용 — 반드시 확인 후 발송

### Step 8 — Outlook에 삽입

- **[Outlook 삽입]** 버튼 클릭
- Outlook 답장 창이 자동으로 열리고 초안이 삽입됩니다
- 참조 페이지 이미지를 선택한 경우 이메일 본문에 이미지로 함께 삽입됩니다

### Step 9 — 최종 검토 후 발송

- Outlook에서 내용을 최종 확인하고 **직접 발송**
- 이 도구는 초안 생성만 지원하며, 자동 발송은 하지 않습니다

## 7. 분석 모드 선택

| 모드 | 속도 | 권장 상황 |
|---|---|---|
| **자동** | 중간 | 대부분의 경우 (기본값) |
| **텍스트 전용** | 빠름 | 빠른 답변이 필요하거나 텍스트만으로 충분한 경우 |
| **PDF 직접 분석** | 느림 | 도면, 표, 그래프 등 시각 정보가 핵심인 경우 |

### 자동 모드 동작 방식

1. 텍스트 매뉴얼로 1차 분석
2. 시각 정보가 필요하다고 판단되면 자동으로 PDF 분석으로 전환
3. 최종 결과 반환

> 대용량 PDF(900페이지 초과)는 관련 페이지만 자동 추출하여 처리합니다.

## 8. 추가 지시사항 활용

메인 화면 우측 **[추가 지시사항]** 텍스트 박스에 자유롭게 입력합니다.

### 활용 예시

```
언어/형식 관련:
- 답변을 영어로 작성해주세요
- 번호 목록 형식으로 단계별 안내를 작성해주세요
- 전문 용어를 그대로 사용해주세요

내용 관련:
- 고객이 초보자이므로 최대한 쉽게 설명해주세요
- 이 문제는 소프트웨어 설정 문제로 판단됩니다
- 구체적인 파라미터 값 예시를 포함해주세요

톤/긴급도 관련:
- 고객이 긴급한 상황이므로 먼저 공감 표현을 포함해주세요
- 격식 있는 어투로 작성해주세요
```

> 추가 지시사항은 **기본 답변 원칙보다 우선 적용**됩니다.

## 9. 참조 페이지 기능

초안 생성 시 PDF를 분석한 경우, 이메일 초안 위쪽에 **참조 페이지 목록**이 나타납니다.

### 페이지 미리보기

- 각 항목 옆 **[미리보기]** 버튼 클릭
- 새 창에서 해당 PDF 페이지가 이미지로 표시됩니다

### 페이지 선택 및 이메일에 첨부

1. 이메일에 포함할 페이지의 **체크박스** 선택
2. **[Outlook 삽입]** 버튼 클릭
3. 선택한 페이지들이 이메일 본문에 **인라인 이미지**로 삽입됩니다

> 텍스트 전용 모드로 분석한 경우 참조 페이지 목록은 표시되지 않습니다.

## 12. 자주 묻는 질문 / 문제 해결

### Q1. "Outlook에서 메일을 찾을 수 없습니다" 오류 / "서버 실행이 실패했습니다" 오류

**원인 A:** Outlook이 실행되지 않았거나, 메일이 선택/열려있지 않음

**해결:**
1. Outlook을 실행
2. 답변할 메일을 클릭하여 선택하거나 열기
3. 다시 **[메일 불러오기]** 클릭

**원인 B:** 프로그램과 Outlook의 실행 권한이 다름 (UAC 불일치)

Windows에서 프로그램과 Outlook의 관리자 권한 수준이 다르면 COM 연결이 차단됩니다.

**해결:** 둘 다 **일반 권한(관리자 아님)** 으로 실행하는 것이 정상입니다.
- `OutlookGemini.exe`를 일반 더블클릭으로 실행 (우클릭 → "관리자 권한으로 실행" 사용 금지)
- Outlook도 일반 더블클릭으로 실행
- 만약 Outlook이 관리자 권한으로 실행된 경우 → Outlook 종료 후 일반 권한으로 재시작

### Q2. 매뉴얼이 매칭되지 않음

**원인:** 경로 설정 오류 또는 txt 매뉴얼 파일 없음

**해결:**
1. 설정 > **[매뉴얼 관리]** 탭에서 폴더 경로 재확인
2. txt 매뉴얼 폴더에 `.txt` 파일이 실제로 있는지 확인
3. PDF만 있고 txt 변환이 안 된 경우 → [PDF → txt 변환](#5-3-pdf--txt-일괄-변환) 진행

### Q3. "API 키 오류" 또는 초안 생성 실패

**원인:** 잘못된 API 키, 인터넷 연결 불가, API 할당량 초과

**해결:**
1. 설정 > **[Gemini API 키]** 탭에서 **[연결 테스트]** 버튼 클릭
2. 인터넷 연결 확인
3. [Google AI Studio](https://aistudio.google.com/) 접속 → 할당량 확인 또는 새 키 발급

---

### Q4. 초안 생성이 매우 느림

**원인:** 대용량 PDF 분석 중

**해결:**
- **텍스트 전용** 모드로 변경하여 빠르게 생성
- **자동** 모드 사용 시 시각 정보가 불필요한 문의는 텍스트로 처리됩니다

### Q5. Outlook 삽입 후 이미지가 보이지 않음

**원인:** Outlook의 이미지 자동 표시 설정 문제

**해결:**
1. Outlook > 파일 > 옵션 > 보안 센터 > 자동 그림 다운로드 설정 확인
2. 답장 창 상단의 "그림 표시" 클릭

---

### Q6. PDF 변환 후 내용이 비어있음

**원인:** 스캔 이미지 기반 PDF (텍스트 레이어 없음)

**해결:**
- Adobe Acrobat 또는 OCR 도구로 텍스트 레이어 생성 후 재변환
- 또는 **PDF 직접 분석** 모드 사용 (Gemini가 이미지로 직접 읽음)

### Q7. 프로그램이 갑자기 종료되거나 응답하지 않음

**해결:**
1. 프로그램을 재시작
2. Outlook이 정상 실행 중인지 확인
3. 인터넷 연결 상태 확인
4. 문제가 반복되면 프로그램 폴더의 `config.ini` 파일을 확인하여 경로 오류가 없는지 점검

### Q8. 초안의 [추정] / [확인 필요] 태그 의미

| 태그 | 의미 | 조치 |
|---|---|---|
| `[추정]` | 매뉴얼에 없어 일반 기술 지식으로 보충한 내용 | 내용 검토 후 확인 |
| `[확인 필요]` | 불확실하거나 매뉴얼에서 확인 불가한 내용 | 반드시 기술 검토 후 발송 |

---

### Q9. 설정값을 직접 편집하고 싶음

프로그램 폴더의 `config.ini` 파일을 메모장으로 열어 직접 수정할 수 있습니다.

```ini
[gemini]
api_key = AIzaSy...          ← Gemini API 키

[paths]
manual_dir = C:/경로/txt폴더  ← 텍스트 매뉴얼 폴더
pdf_dir = C:/경로/PDF폴더     ← PDF 원본 폴더

[settings]
max_manual_context = 80000   ← 매뉴얼 참조 최대 글자 수

[company]
role = 국내 공식 대리점       ← 회사 역할 설명
```

> 경로 구분자는 `\` 대신 `/` 를 사용하세요.
> 저장 후 프로그램을 재시작하면 적용됩니다.
